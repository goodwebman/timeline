# React + TypeScript + Webpack Starter

Простой стартовый шаблон для проектов на **React** с **TypeScript**, сборкой через **Webpack**

---

## ⚡ Быстрый старт

### 1. Установка зависимостей

```bash
npm install
# или
yarn install
```

### 2. Локальный сервер разработки

```bash
npm run dev
# или
yarn dev
```
